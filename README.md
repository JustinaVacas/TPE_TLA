## Gramática

* Permite asignar variables de tipo texto:
```
texto [variable] = [valor]
num [variable] = [valor]
```

* Permite imprimir por pantalla:
```
print [variable]
```
* Permite traducir la palabra sea a texto o a braille:
```
traducir [variable]
```
